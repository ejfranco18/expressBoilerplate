# expressBoilerplate

Express boilerplate with additional endpoints for movies and movie categories, integrated with VueJS.

## Install It
```
npm install
```

## Run It
#### Run in *development* mode:

```
npm run dev
```

## Install Front

```
cd front
npm install
```

## Run front
#### Run:

```
npm run serve
```

#### Run in *production* mode:

```
npm compile
npm start
```

#### Deploy to the Cloud
e.g. CloudFoundry

```
cf push expressBoilerplate
```

### Try It
* Point you're browser to [http://localhost:3000](http://localhost:3000)
* Invoke the example REST endpoint `curl http://localhost:3000/api/v1/examples`

## APIs
#### Categories:
* Get all categories
* Get category
* Post category
* Put category
* Delete Category

#### Movies:
* Get all movies
* Get movie
* Post movie
* Put movie
* Delete movie
